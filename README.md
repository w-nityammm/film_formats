## Local Setup

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

*Copy any default video file inside the `public` directory as `demo.mp4` to auto-load it on startup.*

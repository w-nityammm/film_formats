# Film Format Explorer

A minimal react app to experience different cinematic film formats with your own custom footage, right in your own browser, no servers involved. Inspired by odysseymovie.com.

## Local Setup

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

*Copy any default video file inside the `public` directory as `demo.mp4` to auto-load it on startup.*
